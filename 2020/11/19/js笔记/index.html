10.标识符

/*
 * 标识符
 *  在js中所以的可以有我们自主命名的都可以称为标识符
    列如；变量，函数名，属性名都属于标识符
 * 命名一个标识符时需要遵守如下规则，
     1. 标识符中可以有数字 _ $，
     2.标识符不能以数字开头，
     3.标识符不能是JS中的关键字或保留字；
     4.标识符一般都采用驼峰命名法
         首字母小写，每个单词开头字母大写，其余字母小写；

3.number 数值

6.强制类型转换

/*
  强制数据类型转换
     指将一个数据类型转换为其他数据类型；
      转为 string     number     
  调用被转换类型的tostring（）方法
  给调用方法不会影响到变量， 他会将转换的结果返回；
  该方法不能转null    undefined 因为他们没有tostring方法
  推荐使用 变量名=string(变量名)；函数
 */

7.转换为number

/* 
  使用 变量名=number(变量)函数；
  * 字符串转数字
      纯数字转换为数字
      如果有非数字内容转换为nan
      字符串是空串或是空格为0
   * 布尔转数字
      true转为1
      false转为0
   * null转为数字为0
   * undefined转数字 nan
 */

8.parseint

/*
   这种方法专门用来对付字符串
   parsenint()把一个字符串转换为一个整数，可将字符串中有效内容去出来；只会去整数
   
   parsefloat()把一个字符串转换为一个浮点数 和上方作用类似可获得有效小数

 */

9.怎么表示进制数

/*
  在js中如果需要表示16进制数，则要以0x开头；
              表示8进制数，则要0开头；
              表示2进制数，则要0b开头；
 */

10.转换为boolean

/*
  将其他数据类型转换为boolean
      使用Boolean();
  * 数字转布尔
       除了0和nan，其余都是true
  * 字符串转布尔
       除了空串其余都是true
  * null和undefined都是false
  * 对象也是true
 */
